# ClickFit

Effortless window management for macOS. Drag, snap, tile, and resize — all through your mouse.

## Installation

Download the latest `.dmg` from the [Releases](https://github.com/mertogzhn/ClickFitMac/releases) page, open it, and drag **ClickFit.app** to `/Applications`.

## Requirements

- macOS 15 (Sequoia) or later
- Accessibility permission (granted at first launch via System Settings → Privacy & Security → Accessibility)

## Features

- **Mouse gestures** — drag to move, right-drag to resize, double-click to center or maximize
- **Smart snapping** — windows snap to screen edges, other windows, and grid positions
- **Synchronized resize** — couple adjacent windows to resize them together
- **Tile layouts** — 20+ built-in multi-window arrangements + custom user-drawn layouts
- **Radial layout picker** — modifier + right-click to pick a layout in one gesture
- **Throw-to-zone** — directional gesture to tile windows by throwing them
- **Keyboard shortcuts** — customizable global hotkeys for every action
- **Multi-display aware** — move windows between monitors and virtual desktops
